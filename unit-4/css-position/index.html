Static

static is the default value. An element that is positioned static; is not positioned in any special way.

Relative

Relative behaves the same as static unless you add some extra properties like top, right, bottom, left
A fixed element is positioned relative to the viewport, which means it always stays in the same place EVEN if the page is scrolled. As with relative, the top, right, bottom, and left properties are used.

position: absolute;

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed):
